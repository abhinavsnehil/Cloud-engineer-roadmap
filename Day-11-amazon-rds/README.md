## What is Amazon RDS
- Managed relational database service
- AWS handles patching, backups, and maintenance
- Reduces operational overhead

## Supported Engines
- MySQL
- PostgreSQL
- MariaDB
- Oracle
- SQL Server
- Amazon Aurora

## High Availability
- Multi-AZ deployment provides automatic failover
- Standby instance is created in another AZ

## Performance & Scalability
- Read replicas improve read performance
- Vertical scaling supported

## Backup & Recovery
- Automated backups
- Manual snapshots
- Point-in-time recovery

## Security
- Runs inside VPC
- Controlled using Security Groups
- Data encryption at rest and in transit

## Key Takeaways
- RDS simplifies database management
- Multi-AZ ensures high availability
- Read replicas improve scalability
