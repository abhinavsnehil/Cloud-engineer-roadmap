## What I Learned
- S3 is an object storage service designed for scalability and durability
- Buckets are globally unique containers for objects
- Objects consist of data, metadata, and a key
- S3 supports multiple storage classes based on access patterns

## S3 Storage Classes
- S3 Standard
- S3 Intelligent-Tiering
- S3 Standard-IA
- S3 One Zone-IA
- S3 Glacier
- S3 Glacier Deep Archive

## Security & Access Control
- Bucket policies
- IAM policies
- ACLs (legacy)
- Server-side encryption (SSE-S3, SSE-KMS)

## Key Takeaways
- S3 provides 99.999999999% durability
- Proper bucket policies are critical for security
- S3 is widely used for backups, static websites, and data lakes
